DataTalksClub - Data Engineering Zoomcamp

January 22, 2023 Â· 1 min | 

These last early mornings and late evenings have been filled with setting up my environment for the Data Engineering Zoomcamp.

My current setup involves a virtual machine, hosting a Postgres database, while being able to query it through pgAdmin - pulling it all up via docker-compose. Furthermore, I got to know the basics on how to set up the Google Cloud infrastructure via Terraform.

Within the next weeks, I will be working on workflow orchestration via Prefect, data warehousing in BigQuery, analytics engineering with DBT, and batch and stream processing with Kafka. The cherry on top will be the final project, where everything comes together in a full greenfield project.

Thanks to all course teachers for the great intro and support - I am very much looking forward to the next phases.
